What is Web Development?

Web development is a very important category in the field of programming. Every website you visit every single day is thanks to web developers. As a matter of fact, this website you're looking at right now wouldn't be possible without the help of the most basic website development tool, HTML. HTML (HyperText Markup Language), although not technically a "programming" language, is the basic building block for putting content on the web.

As mentioned above, the main technology used for front end development is some variation of HTML. The backend of website development is when things get interesting. In a post found online here the author provides a list of the top 15 back end programming languages. He also speaks more about the benefits of each one, if you are interested.

  1. JavaScript
  2. Java
  3. Python
  4. CSS
  5. PHP
  6. Ruby
  7. C++
  8. C
  9. Shell
  10. C#
  11. Objective C
  12. R
  13. VimL
  14. Go
  15. Perl
